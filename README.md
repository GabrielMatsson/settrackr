# TDDD27_2026_SetTrackr


### The idé and functionalities
Gym web app where you can track your workouts with set and rep tracking. The user inputs each workout with name, weight, reps, sets. This get stored and you can view your progress on a statistics page. You can also have friends and track their progress. There are many apps like this but they are all very expensive and behind a paywall so I want to make something that I would use personally.

### TeckStack
Client-side
- Next.js
- Tailwind css
- Auth.js (google login)

Server-side
- FastAPI
- Swagger (endpoint testing)
- JWT verification
- PostgreSQL (via Supabase)
